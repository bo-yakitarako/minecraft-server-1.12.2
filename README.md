# みねくらの1.12.2のさーばーだよ
`mods`フォルダと`config`フォルダだけ追跡して、MODの変更に簡単に対応するよ

## やり方
- どこでもいいのでこのリポジトリを`clone`する
```
git clone https://github.com/bo-yakitarako/minecraft-server-1.12.2.git
```
- Minecraftの1.12.2をいったんプレイして、起動したら閉じる
